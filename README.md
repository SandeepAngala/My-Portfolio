# Sandeep Angala — Personal Portfolio

A premium, minimalistic personal portfolio website built with vanilla HTML, CSS, and JavaScript.

![Light & Dark Mode](https://img.shields.io/badge/Theme-Light%20%26%20Dark-556B2F)
![Responsive](https://img.shields.io/badge/Responsive-Mobile%20First-6B8E23)

## ✨ Features

- **Premium Minimal Design** — Cream & Olive Green color palette
- **Light / Dark Mode** — Smooth toggle with localStorage persistence
- **Fully Responsive** — Mobile-first design with fluid typography
- **Scroll Reveal Animations** — Elements animate into view on scroll
- **Typing Animation** — Dynamic hero tagline
- **Interactive Skill Cards** — Hover effects and progress indicators
- **Contact Form** — Client-side validation with feedback
- **Sticky Navbar** — Glass-morphism blur with active section highlighting

## 🚀 Getting Started

### Open Directly
Simply open `index.html` in any modern browser.

### With a Local Server
```bash
npx -y serve .
```
Then open `http://localhost:3000` in your browser.
